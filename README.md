## Install:
1. npm install
2. npm run dev

## Get filtered image:
  ```
    curl "http://localhost:8082/filteredimage?image_url=https://upload.wikimedia.org/wikipedia/commons/b/bd/Golden_tabby_and_white_kitten_n01.jpg" --output filtered.jpg
  ```
